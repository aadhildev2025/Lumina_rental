# LUMINA | Premium Rental Properties

![Lumina Header](https://images.unsplash.com/photo-1512917774080-9991f1c4c750?auto=format&fit=crop&w=1200&q=80)

Lumina is a sleek, modern, and high-performance web application designed for premium home rentals in Sri Lanka. It provides a curated selection of apartments, villas, and commercial spaces with a focus on luxury and ease of use.

## ✨ Key Features

- **Premium Property Catalog**: Browse through high-end apartments, hillside villas, and minimalist studios.
- **Dynamic Inquiry System**: Integrated WhatsApp concierge for personalized property curation and support.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices with a smooth mobile navigation experience.
- **Modern Aesthetics**: Built with a dark-themed, glassmorphic UI using high-quality imagery and smooth animations.
- **Fast Performance**: Powered by Vite and React for ultra-fast load times and seamless transitions.

## 🛠️ Tech Stack

- **Framework**: [React.js](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd "Home Rental"
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:5173](http://localhost:5173) in your browser.
